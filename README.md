A simple Vue.js catalogue of cooking workshops. Browse courses, see availability, and save favorites to your wishlist.  

---

## Features
- Dynamic course cards with title, chef, price, and skill level  
- Shows availability or "Sold Out" status  
- Save courses to a wishlist with a counter  
- Responsive design for desktop and mobile  

---

## Quick Start
1. Install dependencies:
npm install

### Compiles and hot-reloads for development
